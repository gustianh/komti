Tentang Website Ini

Website ini sebenarnya sudah lama ingin saya buat karena saya telah selesai belajar Gratis dan beasiswa dari penyedia kelas online Dicoding dan BuildWithAngga. Saya ingin produk nyata sehingga apa yang saya dapat bisa di implementasikan. Bahkan untuk foldernya pun sudah saya buat sejak Februari 2021 di GitHub saya. Desain sudah ada, kebetulan untuk WireFrame saya pakai Whimsical. Kemudian setelah proses pembuatan WireFrame selesai, saya mulai membangunnya menggunakan CSS Framework Bootsrap. Oh iya, mungkin terlihat kurang professional ya karena saya tidak melanjutkan ketahap desain website menggunakan tools semacam Figma atau Adobe XD. Ya saya sudah tidak sabar ingin segera mewujudkannya. Apalagi makin saya terjun ke bidang Front - End ini, saya makin tertantang. Impian saya tambahkan, saya ingin website saya bisa diakses oleh orang lain dan turut berkontribusi memberi insight bagi mahasiswa baru Universitas Pakuan khususnya Ilmu Komputer.
