Tentang Website Ini

Banyak mahasiswa butuh nomor dosen untuk keperluan mereka dan saya juga sering ditanya, "Eh Gus, sidang itu ngapain aja sih? OBS itu ngapain sih?". Nah, website ini terinspirasi dari pertanyaan - pertanyaan tersebut. Saya akhirnya berpikir untuk menyediakan platform untuk mengenalkan berbagai macam sidang. Saya sangat antusias dan tertantang dalam membuat proyek ini karena kebetulan saya sedang belajar Front - End. Sebagai langkah awal dan agar lebih terskema dengan baik saya merancang WireFrame terlebih dahulu. Untuk WireFrame saya pakai Whimsical. Untuk desain ilustrasi saya mengambil dari UnDraw. Hmm, kenapa UnDraw? Karena gratis stocknya. Selanjutnya saya mulai membangunnya menggunakan CSS Framework Bootstrap dengan durasi pengerjaan 1 bulan, dan dikerjakan setiap hari selama 1 jam. Pengerjaan tanpa adanya UI/UX Design membuat pengerjaannya memakan waktu yang lama. Saya sadar bahwa apa yang saya lakukan agak menyalahi rule Front - End. Tapi yang saya ingin tonjolkan adalah konten dan niat saya untuk sedikit memberi kemudahan bagi teman - teman semua. At the end, Semoga platform ini bermanfaat. Just so you know, I will improve this platform A.S.A.P!
