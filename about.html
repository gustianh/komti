Tentang Website Ini

Website ini terinspirasi dari pertanyaan teman - teman yang diajukan secara pribadi kepada saya. Saya sangat antusias dalam membuat proyek ini karena disamping untuk tantangan saya yang sedang belajar Front-end, tentunya jika website ini bermanfaat bagi orang banyak akan jauh lebih melegakan bukan?. FYI, Bahkan untuk foldernya pun sudah saya buat sejak Februari 2021 lho, di GitHub saya hihi.. Agar lebih terskema dengan baik saya merancang WireFrame terlebih dahulu. Kebetulan untuk WireFrame saya pakai Whimsical. Untuk desain ilustrasi saya mengambil dari UDraw. Hmm, kenapa UDraw? Karena gratis stocknya. Kedepannya, saya akan coba isi dengan hasil karya sendiri biar lebih orisinal kali ya haha.. Doakan saja. Setelah proses pembuatan WireFrame selesai, saya mulai membangunnya menggunakan CSS Framework Bootstrap. Semoga website ini bermanfaat untuk teman - teman semua.
